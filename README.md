## 🚀 Fitur

✅ Tes otomatis parameter umum seperti `id`, `page`, `search`, dll  
✅ Inject payload XSS klasik secara otomatis  
✅ Cek apakah payload muncul di respons (reflected)  
✅ Cocok untuk bug hunter & pentester
